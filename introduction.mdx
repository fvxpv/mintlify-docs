## Key Features

### Moderation
- **Punishment Commands**: Ban, kick, timeout, warn users
- **Moderation Logging**: Track all moderation actions with case IDs
- **Custom DM Messages**: Personalized messages for punishments using the invoke system
- **Warn System**: Auto-punishments based on warning thresholds

### Safety & Security
- **Word Filtering**: Block specific words and phrases with custom punishments
- **Regex Filters**: Advanced pattern matching for content filtering
- **Invite Filter**: Prevent Discord invite links
- **Command Restrictions**: Disable commands for specific users or channels

### Automation
- **Autoroles**: Automatically assign roles to new members (humans and bots)
- **Autoresponder**: Create automatic replies to specific triggers
- **Welcome Messages**: Customizable welcome messages with embeds and variables

### Special Features
- **Starboard**: Highlight popular messages with star reactions
- **Ticket System**: Create support ticket channels with transcripts
- **Giveaways**: Host giveaways with automatic winner selection
- **Invoke System**: Create custom DM templates for moderation actions
